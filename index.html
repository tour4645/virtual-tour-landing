สัมผัสประสบการณ์ความจริงเสมือน (VR) และความจริงเสริม (AR) ที่ไร้รอยต่อ เปลี่ยนการเรียนรู้แบบเดิมให้กลายเป็นโลกที่สัมผัสได้จริง

Project Overview

โปรเจกต์ Virtual Tour คือการผสานเทคโนโลยี VR & AR เข้ากับงานวิศวกรรมและการศึกษา โดยใช้ Unity ในการสร้าง App ที่สามารถแสดงผลโมเดล 3D แบบโต้ตอบได้

เราออกแบบมาเพื่อใช้ร่วมกับชุดฝึก โดยผู้ใช้สามารถส่องดูการทำงานของอุปกรณ์ผ่าน AR เพื่อทำความเข้าใจหลักการทำงานเชิงลึกที่ไม่สามารถมองเห็นได้ด้วยตาเปล่า

Core Capability

  • ✅ AR Model Overlay System
  • ✅ Interactive Instruction Set
